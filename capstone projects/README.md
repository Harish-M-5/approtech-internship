# Title:
🌍 GlobeXchange – Smart Currency Converter

---

## Introduction:
GlobeXchange is a smart currency converter web application that converts one country's currency into another using live exchange rates. It provides a simple, fast, and user-friendly interface.

---

## Overview & Features:
- Live currency conversion
- User-friendly interface
- Responsive design
- Currency swap option
- Supports multiple world currencies
- Fast conversion
- Java backend API integration

---

## Use Case:
- Travelers
- Students studying abroad
- International business
- Online shopping
- Currency exchange calculations

---

## Problem Solved:
- Eliminates manual currency calculations
- Provides quick and accurate conversion
- Uses live exchange rates
- Easy to use for everyone

---

## Technologies Used:

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Core Java
- Java HTTP Server
- REST API

---

## How to Run:

1. Install Java JDK
2. Open backend folder
3. Compile Java files
4. Run Main.java
5. Open index.html using Live Server
6. Click Convert

---

## How to Connect Frontend and Backend:

1. Start Java backend on Port 8080
2. Open frontend in browser
3. Frontend sends request using Fetch API
4. Backend processes request
5. Converted result is displayed

---

## Conclusion:

GlobeXchange is a simple and efficient currency converter developed using HTML, CSS, JavaScript and Core Java. It provides accurate currency conversion with a responsive user interface and demonstrates full-stack web development concepts.
